# DisenoSoftware-Entrega-final
Para levantar la aplicación:
- Crea un .env en la raíz del proyecto siguiendo la estrucutra de .env.example
- Docker debe estar corriendo en la computadora
- Abrir terminal en la raíz del proyecto (donde está el docker-compose) y ejecutar docker-compose --env-file .env up -d --build
- Dirigirse a http://localhost:3000/
